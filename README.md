# MVVM News App

A News App built using MVVM Architecture
## Features

- Display Top-headlines
- Search news on any topic
- Save any news article
- Delete Saved Articles
- Share News 
- View news article in chrome tab intent


## Libraries and Dependencies

- Dagger Hilt
- Retrofit
- Flow Live Data
- Coroutines
- Material Design
- View Models
- Recycler View
- Room Dependencies
- KTX
